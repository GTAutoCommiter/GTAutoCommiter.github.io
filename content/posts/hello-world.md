# 你好，世界！

欢迎来到我的第一篇博客。这个网站是使用 **Next.js 14**、**Tailwind CSS** 和 **TypeScript** 构建的。

## 为什么要建这个网站？

作为开发者，我希望能有一个分享想法和展示作品的空间。这个网站具备以下特点：

- 🚀 **Next.js App Router**：提供极致的性能和开发体验。
- 🎨 **Tailwind CSS**：轻松实现响应式和精美布局。
- 🌓 **深色模式**：更好的阅读体验。
- 💬 **Giscus**：利用 GitHub Discussions 实现的无服务器评论系统。
