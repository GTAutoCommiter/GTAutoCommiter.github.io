# 初始化相关

## Q：如何在项目中引入jest？

* A：在开发依赖中引入jest

  ````bash
  //yarn环境
  yarn add jest --dev
  
  //npm环境
  npm instail jest -D
  ````

## Q：如何使用？

* A：

  1. 先创建逻辑文件`xxx.js`

  2. 创建测试用例文件`xxx.test.js`

  3. 跑测试用例

     ````bash
     //yarn 环境
     yarn test
     //npm 环境
     npm start test
     ````

     > PS：注意在执行之前，检查package.json配置脚本设置
     >
     > ````json
     > "scripts": {
     >   "test": "jest"
     > },
     > ````
