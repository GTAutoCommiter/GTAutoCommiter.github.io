## String

### Q：复制字符串并能够填充空格满足长度条件的方法有哪些？

* A：

  | 方法                          | 描述           |
  | ----------------------------- | -------------- |
  | `padStart(length,replaceStr)` | 往前补填充字符 |
  | `padEnd(length,replaceStr)`   | 往后补填充字符 |

  * `length`：长度
    * 参数小于字符串的长度，则不会添加填充。
  * `replaceStr`：填充字符

## `eval()`方法

### Q：能干嘛？

* A：执行 ECMAScript(JavaScript)字符串。

### Q：怎么用？

* A：

  ````javascript
  eval("console.log('hi')");
  //上面这行代码的功能与下一行等价:
  console.log("hi");
  ````

  * 参数：需要执行的 ECMAScript(JavaScript)字符串。
