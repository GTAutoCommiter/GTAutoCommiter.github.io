Css使用技巧

Q：如何对png进行换色处理？​

  • A：对目标png元素设置css样式

    //其本质就是生成一个新的阴影，改变阴影的颜色，将阴影移到原png位置
    .img {
        transform: translateX(-100%);
      	filter: drop-shadow(120px 0 {想要换的颜色});
    }

  • 参考：

    【掘金】用CSS也可以任意改变PNG图标颜色？

Q：flex布局下如何实现页面居中？​

  • A：

    • CSS

      .father{
        display:'flex';
        align-item:'center';
        justifyContent:'center';
      }
      .child{
        width:'50vw';
        height:'50vw';
      }

    • HTML

      <div class='father'>
        <div class='child'>子盒子</div>
      </div>
