版本：v17

位置：`packages/react-reconciler/src/ReactFiber.js`

目的：根据 React 元素的类型和属性创建一个新的 Fiber 节点。

参数：

| 键名         | 类型       | 描述                     |
| ------------ | ---------- | ------------------------ |
| type         | Any        | React 元素的类型         |
| key          | string     | 元素的唯一键。           |
| pendingProps | Object     | 该元素的待处理属性。     |
| Owner        |            | 元素的所有者，用于调试。 |
| mode         | TypeOfMode | fiber 应该以何种模式运行 |
| lanes        | Lanes      | 工作的优先级             |
