版本：v17

位置：`packages/react/src/jsx/ReactJSXElement.js`

目的：专门用于在生产环境中创建React元素

参数：

| 键名     | 类型                              | 描述               |
| -------- | --------------------------------- | ------------------ |
| type     | 字符串或一个React组件（类或函数） | 元素类型           |
| config   | 一个包含属性和特性的对象          | 元素配置           |
| maybeKey | 字符串                            | 可能包含元素的 key |
