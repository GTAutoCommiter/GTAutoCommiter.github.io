版本：v17

位置：`packages/react-reconciler/src/ReactFiber.js`

目的：用于创建一个新的 Fiber 节点。

参数：

| 键名         | 类型   | 描述                                      |
| ------------ | ------ | ----------------------------------------- |
| this         | object |                                           |
| tag          | string | 表示 Fiber 的类型，例如函数组件、类组件等 |
| pendingProps |        | 新的 props                                |
| key          | string | 用于标识 Fiber 的唯一性                   |
| mode         | string |                                           |
