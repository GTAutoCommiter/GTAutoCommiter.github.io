    * 【注意】React 要求永远不要直接修改 `state`，而是在 state 发生改变时始终使用 `this.setState()`。

      ````jsx
      //【错误示例】

      ````

    * React 可以批量处理多个 state 更新以提高性能。（即：通过 `setState` 方法进行的 state 更新可以是异步的）
